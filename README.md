# Arova Technologies

> Enterprise-grade business automation and high-performance web infrastructure.

Arova Technologies is a specialized engineering and growth team architecting digital ecosystems that eliminate friction. We build custom WhatsApp chatbots for clinics, AI call assistants, and high-performance, scalable websites.

![Arova Technologies](https://arovatechnologies.vercel.app/og-image.jpg)

## 🏗️ Core Architecture & Services

* **Custom Operating Systems:** End-to-end platforms tailored for specific industry workflows (e.g., Arova Healthcare).
* **Workflow Automation:** Seamless integration of AI call pickers, WhatsApp routing, and smart scheduling APIs.
* **Web Infrastructure:** Blazing-fast, SEO-optimized frontends built for maximum conversion and reliability.

## ⚡️ Tech Stack

This project is built with a modern, high-performance frontend stack:
* **React 18** - UI Library
* **Vite** - Next Generation Frontend Tooling
* **Tailwind CSS** - Utility-first CSS framework
* **React Router 6** - Client-side routing
* **Web3Forms** - Serverless form handling

## 🚀 Local Development

To get this project running locally on your machine:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/devdeepsaha/arova-technologies.git](https://github.com/devdeepsaha/arova-technologies.git)
    ```

2.  **Navigate into the directory:**
    ```bash
    cd arova-technologies
    ```

3.  **Install dependencies:**
    ```bash
    npm install
    ```

4.  **Start the development server:**
    ```bash
    npm run dev
    ```
    *The site will be available at `http://localhost:5173`*

5.  **Build for production:**
    ```bash
    npm run build
    ```
